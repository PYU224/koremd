# Screenshots Guide

## Required Screenshots

### Phone Screenshots
Place screenshots in:
- `fastlane/metadata/android/en-US/images/phoneScreenshots/`
- `fastlane/metadata/android/ja-JP/images/phoneScreenshots/`

Requirements:
- Format: PNG or JPEG
- Minimum dimension: 320px
- Maximum dimension: 3840px
- Recommended: 1080x1920 (portrait) or 1920x1080 (landscape)
- Quantity: 2-8 images

Recommended screenshots:
1. File list view (empty state)
2. File list view (with files)
3. Editor view (edit mode)
4. Editor view (preview mode)
5. Markdown toolbar in action
6. Settings screen

### Feature Graphic
Place feature graphic in:
- `fastlane/metadata/android/en-US/images/featureGraphic/`
- `fastlane/metadata/android/ja-JP/images/featureGraphic/`

Requirements:
- Format: PNG or JPEG
- Dimensions: 1024x500px (exactly)
- File name: `featureGraphic.png` or `featureGraphic.jpg`

### Icon
Place app icon in:
- `fastlane/metadata/android/en-US/images/icon.png`
- `fastlane/metadata/android/ja-JP/images/icon.png`

Requirements:
- Format: PNG
- Dimensions: 512x512px
- Transparent background (optional)

## How to Take Screenshots

### For Android
1. Open the app on your device or emulator
2. Navigate to the screen you want to capture
3. Press Power + Volume Down (on most devices)
4. Screenshots are saved to Pictures/Screenshots/

### Using Android Studio
1. Open Android Emulator
2. Click the camera icon in the emulator toolbar
3. Screenshots are saved to desktop

### Editing Screenshots
- Use any image editing software
- Remove status bar if needed
- Add borders or shadows (optional)
- Ensure text is readable

## Tips
